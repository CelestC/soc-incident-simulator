# Mini SOC Incident Report Simulator 🛡️

A beginner project simulating a Security Operations Center (SOC) workflow.

## Features
- Simulates suspicious activity detection
- Creates incident reports with steps taken
- Helps understand security monitoring processes

## How to use
1. Run `soc_simulator.py`
2. Follow prompts to simulate suspicious activity and generate report

## Why I built it
To learn about SOC processes and gain practical understanding of incident reporting and response in cybersecurity.
